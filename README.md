# Tic Tac Toe Android App

A simple Tic Tac Toe game developed in Java for Android. This application allows two players to play the classic Tic Tac Toe game on a 3×3 grid, track scores, and reset the game at any time.

## Features

- Two-player gameplay (Player X and Player O)
- Win detection for rows, columns, and diagonals
- Draw detection
- Score tracking for both players
- Reset game functionality
- Simple and user-friendly interface

## Technologies Used

- Java
- Android Studio
- XML (UI Design)
- Android SDK

## Project Structure

- `MainActivity.java` – Handles game logic, player turns, score tracking, and win detection.
- `activity_main.xml` – Defines the user interface layout.
- Unit and Instrumented Test files for basic testing.
